When it comes to sharing your knowledge with others, blogging is considered as one of the best method. But, with blogging comes several challenges to be faced:
1. Buying Domain & Hosting
2. Designing the website with HTML & CSS (Or maybe Wordpress XD )
3.  Writing and Uploading Posts (The Hardest Part XD)

The 3rd one is the hardest because I won't help you on that! But what I will help you with is the 1st and the 2nd one.

------------

> # Github! The Hatem Tai!

------------
Well Well! Github is the solution of our first challenge. It let you host one static website for completely free. If you don't know what github is, It is a place where people share their codes. It is built for colaborative developments. You can find more information [here](https://www.howtogeek.com/180167/htg-explains-what-is-github-and-what-do-geeks-use-it-for/ "here").
> When signing up for github, you will set an username. If you open a repository with the name, `username.github.io`. BOOM! You got your free website hosted. As an example, my username is `dstushar7`. You came here by `dstushar7.github.io` 

If you upload an index.html file in this repository, that html file will be hosted on `username.github.io` as the homepage. 
So, it means we have somewhere to host our static website, for free!

# Jekyll The Magician!

------------
###### What is jekyll?
Jekyll is a static site generator that will build static pages as coded. Jekyll is the most popular static site generator, largely due to its adoption by GitHub. It is written in Ruby. The most useful thing is, it generates the static pages from Markdown files(Textile and Liquid templates too). Editing a Markdown file is quite easy and takes less efforts then HTML(and designing with CSS).  The static pages are then served by a web server. Why we are talking about Jekyll is the engine behind Github. Github's co-founder Tom Preston-Werner is the designer of Jekyll.  More information about the history and purpose of [jekyll can be found here.](https://en.wikipedia.org/wiki/Jekyll_(software) 

------------

I don't want to repeat the process described [ here](https://jekyllrb.com/docs/). This documentation describes each step to setup jekyll locally in your PC. After installing jekyll there are several websites to find jekyll themes.
